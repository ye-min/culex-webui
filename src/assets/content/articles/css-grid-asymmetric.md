对称是默认值。打开任何一个框架的默认样式，你会看到居中的容器、等宽的列、均匀的间距。这些是合理的预设，但也是思维的终点站——大多数页面止步于此。

不对称需要理由。如果你只是随机打破对称，结果会是混乱，不是个性。有说服力的不对称，来自明确的比例关系。

## 黄金比例与 Grid

最经典的不对称比例是黄金比例 1:1.618。在 CSS Grid 里，这可以直接写成：

```css
grid-template-columns: 1fr 1.618fr;
```

这个比例的好处是视觉上"感觉对"，但又不是机械的 1:2。两栏之间有张力，但不失协调。

## 三栏的非对称结构

两栏好理解，三栏的不对称更有趣。一种常见的处理是把中间栏作为视觉重心：

```css
grid-template-columns: 1fr 2.618fr 1fr;
```

两侧是对称的，但整体是非对称的——因为中间栏的权重远大于两侧。这种结构适合以内容为中心的排版，两侧用来放辅助信息或留白。

## 留白也是设计

不对称排版里，留白不是"没放东西"，是主动的设计决策。一栏里的大面积留白会把视线推向另一栏。这种引导比加粗标题更自然。

实践中我发现，12px 的基准网格配合 1.618 的列比，可以覆盖绝大多数排版场景，同时保持足够的设计弹性。
